# 🚑 Smart Ambulance Management System

A web-based Smart Ambulance Management System designed to reduce emergency response time by helping ambulance drivers quickly locate the nearest hospitals and navigate using real-time routing.

🌐 **Live Demo:** https://ambulance-app-14fc3.web.app

---

## 📌 Project Overview

Every second matters during a medical emergency. This project aims to improve ambulance response by providing an intelligent platform that assists ambulance drivers in selecting the nearest suitable hospital based on location and available facilities.

The application uses location services, interactive maps, and cloud storage to make emergency transportation faster and more efficient.

---

## ✨ Features

- 🏥 Hospital Registration Portal

- 🗺️ Interactive Map using Leaflet

- 🛣️ Route Navigation using OSRM
- ⏱️ Estimated Travel Time (ETA)

## 🚀 How It Works

1. Driver opens the application.
2. Current location is detected automatically.
3. Nearby hospitals are fetched from Firebase.
4. Hospitals are sorted based on shortest distance.
5. Driver selects a hospital.
6. The application calculates the fastest route.
7. Navigation and ETA are displayed on the map.
